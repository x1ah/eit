# eit

[![Build Status](https://travis-ci.org/x1ah/eit.svg?branch=master)](https://travis-ci.org/x1ah/eit)

A mini text editor written in Go.

## Install

```
$ go get -u github.com/x1ah/eit
$ eit
```

## Features

- [x] basic editor
- [ ] implement core TUI interface
- [ ] vim hotkey
- [ ] syntax highlight
- [ ] plugins
- [ ] status line
- [ ] auto complete
